# Unemployment Analysis with Python

## About
Analyzing unemployment trends in India and investigating the impact of Covid-19 as part of CodeAlpha Data Science Internship June 2026.

## Libraries Used
- Pandas
- Numpy
- Matplotlib
- Seaborn

## Key Insights
- Highest Unemployment State: Haryana
- Lowest Unemployment State: Meghalaya
- Peak Unemployment Month: May 2020 (Covid-19 Lockdown)
